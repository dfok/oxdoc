LiRE_ContemporaneousSolver :: CheckSolutionSpace ( const model )

Parameters:

Returns:
The dimension of the solution space. It returns -1 if no solution exists, 0 if the solution is unique, or a number greater than 0 than indicates the degree of indeterminacy.
