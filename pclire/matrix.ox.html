 partition

partition ( const X , const aRows , const aColumns )
Partitions a matrix
Parameters:
X the matrix to partition
rows vector, array or number containing the desired number of rows for each block, except the last one. Specify an empty array to request all rows. Specify negative numbers to ignore a block row.
columns vector, array or number containing the desired number of columns for each block, except the last one. Specify an empty array to request all columns. Specify negative numbers to ignore a block column.
Returns:
An array of submatrices, row-by-row.
Example:
X = unit(4);
[A, B, C, D] = partition(X, {2}, {2}); // gives A = C = unit(2), B = D = zeros(2,2)
[E, F] = partition(X, 2, {});          // gives E = unit(2) ~ zeros(2,2), F = zeros(2,2) ~ unit(2)
[G, H] = partition(X, {}, <1>);        // gives G = 1 | zeros(3,1), H = zeros(1, 3) | unit(3)
