 LiRE_Solution : LiRE_BaseModel

Represents a solution to a LiRE_Model model. The solution is of the form:
State space: \(\Theta_y(0) \mathbf{y}_t = c_y + \sum_{i=1}^k \Theta_y(i) \mathbf{y}_{t-i} + \sum_{i=0}^l \Theta_z(i) z_{t-i} + \Theta_\varepsilon M_1 \varepsilon_t + \Theta_\zeta \zeta_t\),
Observation: \(y_t = A\mathbf{y_t}\)
Driving process: \(z_t = c_z + \Phi z_{t-1} + \varepsilon_t\), \(\varepsilon_t\) exogenous, i.i.d.
Sunspot process: \(\zeta_t\) exogenous.

 Generate

LiRE_Solution :: Generate ( const cT , ... )
Generate a realization of this solution.
Parameters:
cT Desired length of sample
my0 The initial condition for \(\mathbf{y}_t\). The initial conditions are specified in the form \((\mathbf{y}_{-p}, \mathbf{y}_{-p+1}, \cdots, \mathbf{y}_{-1}, \mathbf{y}_0})\), where \(p\) is the number of desired initial conditions. If less initial conditions \(p\) are specified than necessary, the remaining conditions are taken to be \(0\).

Returns:
A DimY() times cT matrix with realizations
Example:
solution.Generate(1000, <>);   // generates 1000 realizations with
                               // initial conditions all set to 0

 ImpulseResponse

LiRE_Solution :: ImpulseResponse ( const cT , const vDirection )
Calculates the impulse response function for \(\varepsilon_t\). The impulse response function \(I_\varepsilon(p)\) in the direction \(v\) is given by \(I_\varepsilon(p) = \frac{\partial y_{t+p}}{\partial \varepsilon_t}v.\) The columns of the returned matrix consist of the values of this function for fixed \(v\), for \(p\) running from 0 to cT-1.
Parameters:
cT The number of lags to calculate
vDirection The direction of the impulse. This should be a vector of dimension DimZ().
Returns:
A DimY() times cT matrix
Example:
For a solution to a model with a two-dimensional disturbance term, the following code gives the impulse response of the second component of \(\varepsilon_t\). 
decl IR = solution.ImpulseResponse(30, <0; 1>);
Draw(0, X);
ShowDrawWindow();
