 LiRE_Solver

LiRE_Solver is the base class for LiRE solvers. It contains common functionality for any LiRE solver.

To create and use a new solver, the following steps have to be taken:

  1. Derive a class from LiRE_Solver
  2. Override the methods ClassName, CanSolve, and PerformSolve.
  3. Create a new instance of your newly derived class
  4. Use SetSolver to install this solver into LiRE

virtual LiRE_Solver :: CheckSolutionSpace ( const model )
Checks whether a solution exists and if so, if it is unique.
Parameters:
model a LiRE_Model object that specifies the model to be solved
Returns:
The dimension of the solution space. It returns -1 if no solution exists, 0 if the solution is unique, or a number greater than 0 than indicates the degree of indeterminacy.

 Solve

LiRE_Solver :: Solve ( const model )
Solve the specified model.
Parameters:
model a LiRE_Model object that specifies the model to be solved
Returns:
a LiRE_Solution object that represents the solution.
Comments:
Solve checks whether the inputs are correct, and whether the solver can actually solve the problem. Then, PerformSolve is executed, and its result is checked. The resulting solution is given the name 'Solution to ....', and the variable names are copied from the original model.
