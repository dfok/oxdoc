 LiRE_BaseModel

Representation of an abstract model in an \(n\)-vector valued variable with an AR(1) disturbance term.

LiRE_BaseModel contains basic functionality for describing the elements of the vector in terms of conditional expectations of other, more basic, variables. This is done by means of "element descriptors". An element descriptor describes one element of the vector \(y_t\) in a LiRE_Model object. So, for an \(n\)-dimensional vector \(y_t\), \(n\) element descriptors are required.

The disturbance term \(z_t\) follows \(z_t = c_z + \Phi z_{t-1} + \varepsilon_t\) with \(\varepsilon_t\) i.i.d. The distribution of \(\varepsilon_t\) can be set as well.

Example:
To describe a 3-dimensional vector \((x_t, x_{t+1|t}, z_{t-1})\), one can use the following code: 
// set up element descriptors
model.SetElementDescriptors({"x", 0, 0}, {"x", 0, 1}, {"z", 1, 0});


println(model.ElementName(0));     // prints "x[t]"
println(model.ElementName(1));     // prints "x[t+1|t]"
println(model.ElementName(2));     // prints "z[t-1]"

 SetElementDescriptors

LiRE_BaseModel :: SetElementDescriptors ( const aElemDesc )
Sets up the variable descriptors
Parameters:
aYnames Array of the form { {name1, i1, j1}, {name2, i2, j2}, ... }. Each element of this array describes one row in the vector. Here, i is the informational lag, and j is the lead. I.e. the variable described is \(E[\mbox{name}_{t+j-i}\ |\ \mathcal{F}_{t-i}]\).

 SetEpsilonDistribution

LiRE_BaseModel :: SetEpsilonDistribution ( const EpsilonDist )
Specifies the distribution for \(\varepsilon_t\).
Parameters:
EpsilonDist an object of a class derived from Distribution. For example, NormalDistribution.
Example:
To set up a one-dimensional standard normal distribution for \(\varepsilon_t\), the following code is appropriate: 
model.SetEpsilonDistribution(new NormalDistribution(0, 1));
