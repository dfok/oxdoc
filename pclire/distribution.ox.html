 Distribution

Abstract class that represents a random distribution. Classes derived from the base class Distribution offer functionality like generating samples. Also, a Distribution has a dimension.
See also:
NormalDistribution, tDistribution, UniformDistribution, EmpiricalDistribution, FixedDistribution
