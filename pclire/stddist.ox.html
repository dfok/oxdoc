 NormalDistribution : Distribution

The multivariate Normal distribution \(\mathcal{N}(\mu, \Sigma)\).
Example:

decl Dist = new NormalDistribution(0, 1);
decl Z = Dist.Generate(20);

 tDistribution : Distribution

The multivariate Student-t distribution \(t_{df}(\mu, \Sigma)\).
Example:
To generate 20 samples from a Student-t distribution with 20 degrees freedom, the following code can be used: 
decl Dist = new tDistribution(20, 0, 1);
decl t = Dist.Generate(20);

 UniformDistribution

UniformDistribution :: UniformDistribution ( vL , vU )
Create a new instance of the UniformDistribution class with range \([l, u]\).
Parameters:
vMin a number or a column vector containing the lower bounds.
vMax a number or a column vector containing the upper bounds.

Comments:
If \(l\) and \(u\) are specified as vectors, the realizations will element-wise be taken from uniform distributions, e.g. \(l = (0, 10), u = (1, 20)\) will generate vectors \((x,y)\) where \(x\) comes from a \(U(0,1)\) distribution and \(y\) comes from a \(U(1,20)\) distribution.
