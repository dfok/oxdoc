LiRE_BackdatedSolver :: PerformSolve ( const model )
Solves the backdated expectations model.
Parameters:
model the model to be solved
Returns:
a solution in the form a LiRE_Solution object or, in case of an error, a LiRE_SolveError object containing an error message and roots if available
Comments:
The model is solved by casting it in a contemporaneous expectations model, and solving the model using LiRE_ContemporaneousSolver. Then, the returned solution is interpreted.
