 SetSolver

LiRE :: SetSolver ( const solver )
Specifies a solver
Parameters:
oSolver This can be either an object derived from LiRE_Solver, or one of the following constants:
SOLVER_AUTOLet LiRE choose a suitable solver
SOLVER_GENERICUse the generic solver without indeterminacy. This solver is based on a predetermined/non-predetermined partition of variables.
SOLVER_SIMSUse the Sims solver. This solver handles indeterminacy, but suitable for less general models.
See also:
LiRE_Solver, LiRE_GenericSolver, LiRE_SimsSolver

LiRE :: Solver ( )
