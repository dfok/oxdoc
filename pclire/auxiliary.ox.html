 relTime

relTime ( i )
Formats a 'relative time' string \(t-i\).
Parameters:
i The lag (if negative) or lead (if positive)
Returns:
If i equals zero, "t" is returned. If i is positive, "t+" plus the lead is returned. If i is negative "t-" plus the lag is returned.

writeCsvMatrix ( const fFile , const mX )
Write a matrix to a file in comma separated value (CSV) format
Parameters:

mX matrix to be exported

 writeMatlabMatrix

writeMatlabMatrix ( const fFile , const sVariableName , const mX )
Write a matrix to a file in Matlab script format
Parameters:
fFile file handle, or file name. If a string is specified, it is regarded as a file name. A new file will be created (any existing file will be overwritten), the matrix is written to the file, and it will be closed again.
sVariableName name of variable to be exported
mX matrix to be exported
Example:
The code 
writeMatlabMatrix("unit.m", "U", unit(4));
results in a file unit.m that looks like 
U = [1 0;
     0 1];
