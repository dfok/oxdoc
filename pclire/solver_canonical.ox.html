 PerformSolve

LiRE_CanonicalSolver :: PerformSolve ( const model )
Solve the model
Returns:
a solution in the form a LiRE_Solution object or, in case of an error, a LiRE_SolveError object containing an error message and roots if available
Comments:
Solving is done in a number of steps:
  • Write the model in the form \(A_c y_{t+1|t} = B_c y_t + Q_c z_t\)
  • Do a Schur decomposition
  • Write the non-predetermined variables as a function of the predetermined variables
